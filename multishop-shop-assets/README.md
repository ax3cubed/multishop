Multishop Shop Assets - Open Source Ecommerce Platform  
=======================================

This library contains all the shop assets, such as themes, ui widgets etc, that are required to render shop store front.
It has to be used together with other `multishop-app` such as `multishop-shop` or `multishop-merchant`.

REQUIREMENTS
------------
* `multishop-kernel`
* `multishop-merchant` or `multishop-shop`
